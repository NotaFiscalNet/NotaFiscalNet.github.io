    TipoCalculo

    Retorna ou define o Tipo de Cálculo do IPI.

    Remarks

    Este campo deverá ser preenchido caso a Situação Tributária indique que deverá informar os dados de tributação.
    Para maiores informações, seja a documentação presente no campo SituacaoTributaria.

    Caso o Tipo de Cálculo seja por Percentual, os campos ValorBaseCalculo e Aliquota deverão ser preenchidos. Caso o Tipo de Cálculo seja por Valor, os campos QuantidadeTotalUnidPadrao e ValorUnidade deverão ser preenchidos.
